You are given an integer array fruits where fruits[i] is the type of fruit the ith tree produces.

You have two baskets and each basket can only hold a single type of fruit. You want to pick fruits from a contiguous subarray such that you can collect the maximum number of fruits.

Example :

Input: fruits = [1,2,1]  
Output: 3
